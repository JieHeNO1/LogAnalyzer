Principle To define error code	"1. Never swallow error information itself
2. Detail the error information as much as you can
3. Never worry error code lost, software will log all the component error code
4. Trouble shotting component error with error code as possible as you can
5. Only define own-developped component error code."	Unnamed: 2
Code Format	"标识码 （错误ERR/状态STA) + 设备 + 子系统 + 部件 + 模块 + 标号
>> 设备 & 子系统 & 部件 & 模块：均三位数字 （其映射表见右边表格）
>> 标号：格式为0xaabbccdd (32位16进制的数字）
>> 设备: MR系统 数字为(002)
>> 子系统：MR 错误码不定义子系统，默认为000
>> 模块：MR 系统未定义模块，默认为000"	例如GA_0x08840000
Component	"Following component needs to define error code:
. Gradient Amplifier
. RF Amplifier
. Table (or Patient Couch)
. SCU
. MMU 
. LCC
. Software (include sequence, reconstruction and software workflow)"	
Error code range	". SCU:                                       0x1000 - 0x1FFF 
. RF Amplifier:                           0x2000 - 0x2FFF
. Gradient Amplifier:                  0x3000 - 0x3FFF
. Table (or Patient Couch):       0x4000 - 0x4FFF
. Coil:                                        0x5000 - 0x5FFF
. Magnet:                                   0x6000 - 0x6FFF
. LCC:                                       0x7000 - 0x7FFF
. Software                                 0x8000 - 0x8FFF "
